# Portfolio — Carlos Manuel Cejas

Static personal site built with plain HTML and CSS.

## Files

- `index.html` — landing page
- `about.html` — personal/about page
- `resume.html` — embedded resume PDF (with link to open the file in a new tab)
- `styles.css` — Bauhaus-inspired styling
- `resume.pdf` — resume asset used by `resume.html`

Deployed on Vercel as a static site.
